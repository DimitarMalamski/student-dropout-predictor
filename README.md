# Student Dropout Prediction

This repository contains a machine learning project focused on early student dropout risk prediction in higher education.

The project explores whether academic, demographic, financial, and administrative indicators can be used to identify patterns related to dropout risk and support early intervention.

## Project Overview

The work is divided into two main notebooks:

- **Student Dropout Predictor**  
  A classification-based analysis comparing multiple machine learning models for predicting whether a student will **drop out** or **graduate**.

- **Student Dropout Risk Scoring with Regression for Early Intervention**  
  A regression-based extension that generates a **continuous dropout-related risk score** between 0 and 1, later expressed as a percentage for easier interpretation.

## Main Topics Covered

- data understanding and exploratory analysis
- data preparation and feature engineering
- model training and comparison
- ethical feature analysis
- threshold tuning
- continuous risk scoring
- calibration and practical interpretation

## Project Files

This repository also includes supporting documentation:

- **Project Proposal**
- **Domain Understanding**
- **Analytic Approach**

These documents explain the problem context, stakeholder perspective, methodological choices, ethical considerations, and the overall structure of the project.

## Goal

The goal of this project is not to make final judgments about students, but to explore how predictive analytics can support the **early identification of students who may need additional attention or support**.

## Notes

- The project uses the **Predict Students’ Dropout and Academic Success** dataset.
- The results should be interpreted as **decision-support indicators**, not as certain predictions.
- Sensitive features were also analyzed separately to examine ethical considerations and more responsible model design.
